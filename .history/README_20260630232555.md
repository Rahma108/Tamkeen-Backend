# 🤖 AI Mock Interview Backend

An AI-powered backend application built with **NestJS** that helps users prepare for technical interviews through ATS resume analysis, AI-generated interview questions, chat and voice mock interviews, and detailed performance reports.

---

## 🚀 Features

- 🔐 JWT Authentication & Authorization
- 👤 User Profile Management
- 📄 CV Upload & Management
- 💼 Job Description Management
- 🤖 AI-Powered ATS Resume Analysis
- ❓ AI Interview Question Generation
- 💬 Chat-Based Mock Interview
- 🎤 Voice-Based Mock Interview
- 📊 AI Performance Reports
- 📈 Dashboard & Interview History
- 👨‍💼 Admin Management
- 📚 Swagger API Documentation

---
